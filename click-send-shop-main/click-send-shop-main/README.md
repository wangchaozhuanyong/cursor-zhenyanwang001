# 大马通 - 马来西亚华人一站式生活服务与合规精选好物平台

基于 React + TypeScript + Vite + Tailwind CSS 构建的前端项目。

## 技术栈

- 框架：React 18 + TypeScript
- 构建工具：Vite
- 样式系统：Tailwind CSS + shadcn/ui
- 状态管理：Zustand
- 路由：React Router
- 数据请求：TanStack React Query
- 表单：React Hook Form + Zod

## 本地开发

```bash
npm install
npm run dev
```

## 打包构建

```bash
npm run build
npm run preview
```
