# 真烟网 - 精选全球好物购物平台

基于 React + TypeScript + Vite + Tailwind CSS 构建的电商购物平台。

## 技术栈

- **框架**: React 18 + TypeScript
- **构建工具**: Vite 5
- **样式**: Tailwind CSS 3 + shadcn/ui
- **状态管理**: Zustand
- **路由**: React Router DOM 6
- **数据请求**: TanStack React Query
- **表单**: React Hook Form + Zod
- **测试**: Vitest + Playwright

## 开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
npm run preview
```
