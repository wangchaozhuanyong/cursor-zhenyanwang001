# 站点功能开关影响矩阵

本文档记录后台“功能开关”的真实影响范围。原则：前端隐藏只是体验优化，核心能力必须由后端接口或业务逻辑同步拦截。

| 开关 | 默认值 | 前端入口影响 | 后端接口影响 | 核心业务逻辑影响 | 测试方法 |
|---|---:|---|---|---|---|
| `mallEnabled` | true | 隐藏分类、搜索、商品详情、购物车、结算等前台商城入口 | 拦截商品列表/详情、购物车、订单预览、订单创建 | 禁止继续浏览商品、加购和下单；历史订单查询不受影响 | 关闭后请求 `/api/products`、`/api/cart`、`/api/orders/preview` 应返回 403 |
| `onlinePaymentEnabled` | true | 隐藏在线支付相关入口 | 拦截支付渠道、支付意图、订单支付、Stripe Checkout | 禁止创建在线支付流程 | 关闭后请求支付接口应返回 403 |
| `pointsEnabled` | true | 隐藏积分入口和积分抵扣 UI | 拦截积分接口；订单预览/创建时拒绝强传积分抵扣 | 不生成积分抵扣，不扣用户积分，最大可用积分为 0 | 关闭后提交 `use_points=true` 应返回“积分功能已关闭” |
| `couponEnabled` | true | 隐藏优惠券入口和下单优惠券 UI | 拦截优惠券接口；订单预览/创建时拒绝强传优惠券 | 不查询、不锁定、不使用用户优惠券 | 关闭后提交 `coupon_id` 应返回“优惠券功能已关闭” |
| `reviewEnabled` | true | 隐藏评价入口 | 拦截评价提交、点赞和后台评价管理 | 禁止新增评价互动 | 关闭后提交评价应返回 403 |
| `inventoryEnabled` | true | 隐藏库存中心入口 | 拦截后台库存管理接口 | 禁止后台库存调整、盘点、组装拆包等库存管理动作 | 关闭后请求后台库存接口应返回 403 |
| `shippingEnabled` | true | 隐藏配送设置入口 | 拦截后台配送模板和设置接口 | 禁止后台修改配送规则 | 关闭后请求配送设置接口应返回 403 |
| `memberLevelEnabled` | true | 隐藏会员等级入口 | 拦截后台会员等级接口 | 禁止后台维护会员等级 | 关闭后请求会员等级接口应返回 403 |
| `customerServiceDownloadEnabled` | true | 隐藏客服/APP 下载页入口 | 暂无核心后端接口 | 不影响订单、商品、用户核心链路 | 关闭后访问前端客服/APP 页面应回到首页 |
| `telegramOrderNotifyEnabled` | true | 控制后台 Telegram 订单提醒设置入口 | 拦截后台 Telegram 设置接口 | 禁止发送或配置 Telegram 订单提醒 | 关闭后保存 Telegram 设置应返回 403 |
| `languageGateEnabled` | true | 前台可拦截非中文浏览器访问 | 暂不拦截 API | 只影响前端展示，不作为安全边界 | 非中文浏览器访问公开页时检查提示，不应破坏 SEO head |
| `trafficAnalyticsEnabled` | true | 隐藏后台流量分析入口，前端停止发送埋点 | 拦截 `/analytics/events` 和 `/admin/reports/traffic` | 禁止采集新埋点和查看流量分析 | 关闭后埋点和后台流量分析接口应返回 403 |

## 暂时隐藏的预留开关

以下开关当前不在后台功能开关页面展示，避免管理员误以为已经完整生效：

- `serviceEnabled`：服务模块尚未形成完整的前端页面、后端接口和业务链路矩阵。
- `restrictedProductComplianceEnabled`：受监管商品合规能力涉及展示、下单校验、SEO、sitemap 等多处规则，未完成完整验收前不作为可随意关闭的开关。

## 验收要求

1. 关闭任一核心开关后，前端入口应隐藏或不可进入。
2. 直接调用相关后端接口必须被拒绝，不能只依赖前端隐藏。
3. 订单预览和订单创建必须保持一致，不能预览允许、下单拒绝，或预览拒绝、下单允许。
4. 开关关闭时返回明确中文错误，方便客服和运营定位。
