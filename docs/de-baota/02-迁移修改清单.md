## 3. 未改动的核心约定（按你的约束）

- **PM2 唯一入口**：`ecosystem.config.cjs`，进程名 `gc-api`，脚本 `src/index.js`。
- **验收**：`deploy/verify-pm2.sh` 仍为强制门禁。

## 4. 服务器侧仍需你执行的操作（非代码）

1. `sudo bash scripts/migrate-from-baota-path.sh`（先 `DRY_RUN=1`）。
2. 安装系统 Nginx，拷贝并启用 `deploy/nginx/site.prod.example.conf`（改域名与证书）。
3. SSL：`certbot --nginx -d YOUR_DOMAIN` 或 acme.sh，并配置续签 cron/timer。
4. `pm2 startup systemd` + `pm2 save`（见 `deploy/systemd/README.md`）。
5. 将 GitHub Secret `DEPLOY_PROJECT_DIR` 改为 `/var/www/click-send-shop`。
6. 验收：`bash deploy/verify-pm2.sh` 通过后再卸载宝塔。
